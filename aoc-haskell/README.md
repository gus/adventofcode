# gus-aoc-haskell

```sh
# example inputs
cat ./inputs/<YYYY>/d<D>.example.<M>.txt | stack run <YYYY>d<D>

# problem input
cat ./inputs/<YYYY>/d<D>.txt
```

Where:

- `<YYYY>` represents the 4-digit year
- `<D>` represents the day's problem
- `<M>` represents some integer `[1..]`, assuming a file exists with that `M`
